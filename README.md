# Bro Code Python Full Course Implementation

This repository serves as a comprehensive collection of practice scripts and projects completed during the **Bro Code Python Full Course**. The curriculum covers the spectrum from fundamental syntax to advanced programming paradigms like Multi-threading and API integration.

## 📖 Curriculum Syllabus

### 1. Programming Foundations
* **Variables & Data Types:** Practical usage of Strings, Integers, Floats, and Booleans.
* **Typecasting:** Explicit and implicit data conversion between types.
* **Mathematics:** Arithmetic operators, built-in functions (`round`, `abs`, `pow`), and the `math` module (`sqrt`, `ceil`, `floor`).
* **User Input:** Handling string-based command line inputs and converting them for logic.

### 2. Logic & Control Flow
* **Conditional Logic:** Implementation of `if`, `elif`, and `else` blocks.
* **Logical Operators:** Managing complex conditions using `and`, `or`, and `not`.
* **Iteration:** Mastering `while` and `for` loops, loop control keywords (`break`, `continue`), and nesting logic.
* **String Slicing:** Efficiently extracting and reversing data from sequences.

### 3. Data Structures (Collections)
* **Linear Collections:** Working with Lists (mutable) and Tuples (immutable).
* **Hash Maps & Sets:** Utilizing Dictionaries for Key-Value mapping and Sets for unique data handling.
* **2D Collections:** Managing matrices and multi-dimensional data grids.
* **List Comprehensions:** Writing concise, Pythonic syntax for generating new collections.

### 4. Modular & Structural Programming
* **Functions:** Defining reusable blocks, parameters, and return statements.
* **Arguments:** Flexible inputs using Positional, Keyword, Default, `*args`, and `**kwargs`.
* **Scope:** Understanding the LEGB rule (Local, Enclosed, Global, Built-in).
* **Modules:** Organizing logic into multiple files and utilizing `if __name__ == "__main__":`.

### 5. Object-Oriented Programming (OOP)
* **Classes:** Creating blueprints for complex data structures.
* **State & Behavior:** Initializing attributes via `__init__` and defining instance methods.
* **Class Variables:** Managing data shared across all instances of a class.
* **Inheritance:** Implementing Single, Multiple, and Multi-level Inheritance.
* **Dunder Methods:** Customizing object behavior for built-in Python operations.
